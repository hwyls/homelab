<h1>Home Lab (Kali Virtual Machine)</h1>
<h2>Description</h2>
This simple project consists of setting up a Kali Linux Virtual Machine on VMWare Fusion running on MacOS. 
<br />


<h2>Languages and Utilities Used</h2>

- <b>Linux Command Line</b> 

<h2>Environments Used </h2>

- <b>Linux</b> (Kali Distribution)
- <b>MacOS</b>
